## 3. Ответы на контрольные вопросы

### 3.1 Учётная запись пользователя

Содержит:

- login  
- UID  
- GID  
- домашний каталог  
- shell  
- пароль (в зашифрованном виде)  
- список групп  

Файлы:

```
/etc/passwd
/etc/shadow
/etc/group
```

Команды:

```bash
id username
cat /etc/passwd
```

---

### 3.2 Основные команды терминала

Навигация:

```bash
cd
pwd
ls
```

Создание и удаление:

```bash
mkdir
touch
rm
```

Права доступа:

```bash
chmod 755 файл
```

История:

```bash
history
```

---

### 3.3 Файловая система

Файловая система — способ организации хранения данных на диске.

Примеры:

- ext4  
- NTFS  
- FAT32  
- XFS  

---

### 3.4 Подмонтированные файловые системы

```bash
mount
df -h
lsblk
```

---

### 3.5 Удаление зависшего процесса

Поиск процесса:

```bash
ps aux
top
```

Завершение:

```bash
kill PID
kill -9 PID
pkill имя_процесса
```
